# ASEE2026-open-source-curriculum
This repository is home to a set of open-source teaching materials used in implementing a Gender-Design Implicit Association Test (IAT) within engineering design curriculum. Details for how to implement and adapt this teaching module can be found in the paper, "Making Implicit Gender Bias Visible in Engineering Design:Developing a Curriculum Module for Implementing a Gender-Design Implicit Association Test", found in the ASEE2026 proceedings.
